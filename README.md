🎬 Movie Information App

📌 Description

This project is a movie information web application that allows users to search for details about any movie by entering its name. It fetches data using an API and displays information such as movie name, release year, director, actors, actress, and songs in a structured format. The project is built to practice JavaScript concepts like API integration, asynchronous programming, and DOM manipulation.

🚀 Features 🔍 Search movie by name 📄 Displays: Movie name with year Director Actors & Actress Songs ⚡ Uses API to fetch data dynamically 🎨 Clean and simple UI

🛠️ Technologies Used HTML CSS JavaScript Fetch API Async/Await

📂 Project Structure Movie-App/ │── index.html │── style.css │── script.js

⚙️ How It Works User enters a movie name Clicks the "Click" button A request is sent to the API Data is received in JSON format Information is displayed on the screen

▶️ How to Run Download or clone the project Open index.html in your browser Enter a movie name Click the button to see results
